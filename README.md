# Ardid — Frontend 

## Descripción
**Ardid** es el frontend de una tienda ficticia inspirada en la idea del internet como una forma de vida: un lugar donde aún es posible escapar, reinventarse y crear universos propios.  
En un mundo donde la inteligencia artificial amenaza con homogeneizar la creatividad humana, **Ardid** resiste. Cada prenda es presentada como un artefacto simbólico, un portal hacia identidades digitales y mundos paralelos creados por quienes todavía imaginan antes de que sea demasiado tarde.
